<h2>Media collection backend</h2>
Link for frontend: https://github.com/konradpieczynski/media_collection-frontend
<br>
<br>
Known bugs:
<br>
Frontend application will crash if backend application is not running.
<br>
<h2>Environment to set:</h2>
APPLE_API_KEY="rapid api key for: https://rapidapi.com/Jfreeg/api/apple-marketing-tools/"
<br>
MOVIE_API_KEY="rapid api key for: https://rapidapi.com/SAdrian/api/moviesdatabase"
<br>
DB_URL="mysql server link"
<br>
MYSQL_USERNAME="mysql username"
<br>
MYSQL_PASSWORD="mysql password"
